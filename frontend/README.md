## Features

* Reusable node abstraction system
* Clean and unified UI design
* Dynamic text node with auto-resizing
* Variable parsing with dynamic handles (`{{variable}}`)
* Backend integration with pipeline parsing
* DAG validation (Directed Acyclic Graph detection)

---

## Tech Stack

* Frontend: React (JavaScript)
* Backend: FastAPI (Python)
* API Server: Uvicorn
